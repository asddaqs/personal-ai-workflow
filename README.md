# Personal AI Workflow
基于Cursor编辑器和OpenCode工具构建的个人编程与办公自动化工作流，用于提升日常开发、学习和办公效率。

## 核心功能
- 🐍 Python脚本自动生成与调试
- 📊 Excel/CSV批量数据清洗与处理
- 📝 技术文档与长文章自动总结
- 💻 代码注释与使用说明自动生成
- 📚 学习资料知识点提取与整理

## 使用工具与模型
- 开发工具：Cursor, OpenCode
- 底层模型：GPT系列, Gemini系列
- 目标：接入小米MiMo大模型进行本地调用测试

## 项目说明
本仓库为个人自用的AI工作流集合，所有代码和脚本均用于个人学习和日常办公场景，**无任何商业用途**。
